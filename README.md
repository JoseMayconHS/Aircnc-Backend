## Construido com ...
- `NodeJs`
  - `express`, administrando as rotas;
  - `cors`, liberando acesso externo;
  - `mongoose`, conectando ao mongoDB (serviço MongoAtlas);
  - `socket.io`, *realtime*;
  - `nodemon`, *realod* automático;
  - `multer`, *upload* de aquivos;
  - `dotenv`, lendo as váriaveis de ambiente.
